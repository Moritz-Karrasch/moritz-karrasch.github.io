2. Allgemeine Hinweise

54 – Das Würfelspiel ist eine App, die sowohl lokal als auch mit einer optionalen Onlinespiel-Funktion genutzt werden kann. Die App kann grundsätzlich ohne klassisches Benutzerkonto verwendet werden. Spielstände, Einstellungen und ähnliche Informationen werden lokal auf Ihrem Gerät gespeichert.

Wenn Sie ausschließlich offline spielen, erfolgt die Nutzung im Wesentlichen lokal auf Ihrem Gerät. Wenn Sie die Onlinespiel-Funktion verwenden, werden zusätzliche technische und spielbezogene Daten verarbeitet, damit Sie einem privaten Spiel beitreten oder ein solches erstellen können.

3. Lokale Datenspeicherung auf Ihrem Gerät

Die App speichert bestimmte Informationen lokal auf Ihrem Gerät in einer SQLite-Datenbank, damit Einstellungen und Komfortfunktionen verfügbar bleiben. Dazu gehören insbesondere:

  • Spielerprofile (Name, Würfelfarbe, Augenfarbe)
  • Spieleinstellungen (Zielpunktzahl, Themes, Sprache)
  • Spielstatistiken (Siege, Niederlagen, gewürfelte Sechsen, Punkte)
  • Würfelfarben-Freischaltungen
  • Premium-Status

Diese lokal gespeicherten Daten werden nicht automatisch an Dritte übermittelt. Durch Deinstallation der App werden alle lokalen Daten gelöscht.

4. Onlinespiel-Funktion

Die App bietet eine optionale Onlinespiel-Funktion. Dabei können Nutzer private Spielräume erstellen oder einem bestehenden Spiel über einen Join-Code bzw. per QR-Code beitreten. Eine öffentliche Spielersuche oder öffentlich einsehbare Spielräume gibt es nicht.

Zur Bereitstellung dieser Funktion können insbesondere der gewählte Spielername, der Join-Code, spielbezogene Einstellungen, Spielstatus, Spielereignisse sowie technisch erforderliche Verbindungsdaten verarbeitet werden. Die Verarbeitung erfolgt ausschließlich, um private Online-Partien technisch bereitzustellen, Teilnehmer zuzuordnen und den Spielablauf zu synchronisieren.

Für die Nutzung der Onlinespiel-Funktion ist kein klassisches Benutzerkonto mit Registrierung erforderlich. Technisch kann jedoch eine anonyme oder temporäre Zuordnung zu einer Spielsitzung erfolgen, damit Spielräume, Spielerstatus und Spielabläufe korrekt verarbeitet werden können.

5. Technische Bereitstellung der Onlinespiel-Funktion

Zur technischen Bereitstellung der Onlinespiel-Funktion können externe technische Dienstleister bzw. Hosting- und Backend-Dienste eingesetzt werden. Diese verarbeiten Daten ausschließlich insoweit, wie dies zur Bereitstellung der Onlinefunktion, zur Datenübertragung, Synchronisierung und technischen Stabilität erforderlich ist.

Sofern dabei personenbezogene Daten verarbeitet werden, erfolgt dies nur im erforderlichen Umfang zur Bereitstellung der App-Funktionalität.

6. Werbung

Die App verwendet Google AdMob, einen Werbedienst der Google Ireland Limited, Gordon House, Barrow Street, Dublin 4, Irland, um Werbeanzeigen innerhalb der App anzuzeigen. Im Rahmen der Werbeanzeigen können Werbepartner über AdMob-Mediation eingebunden werden, darunter Meta Audience Network und Mintegral.

Im Zusammenhang mit der Anzeige von Werbung können Daten verarbeitet werden, zum Beispiel:

  • Technische Geräteinformationen
  • IP-Adresse
  • Werbe-ID (Advertising ID)
  • Informationen zur Interaktion mit Anzeigen

Auf iOS-Geräten wird gemäß den Apple-Richtlinien vor dem Zugriff auf die Werbe-ID eine Einwilligung über das App Tracking Transparency (ATT) Framework eingeholt.

Soweit für bestimmte Verarbeitungen eine Einwilligung erforderlich ist (insbesondere im EWR), erfolgt diese auf Grundlage Ihrer Einwilligung über die Google User Messaging Platform (UMP). Sie können Ihre Einwilligung jederzeit in den App-Einstellungen widerrufen.
